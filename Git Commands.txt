1. git status
2. git init
3. git add (-A to add all files)
4. git commit -m "Add any message"
5. git remote add origin (link)
6. git push origin master

To clone a repository
7. git clone (To download a repository from remote)

To Sync local files with remote repository
8. git fetch --prune origin
9. git reset --hard origin/master
10. git clean -f -d


11. git branch
12. git checkout -b "Name of the Branch"
13. git remote -v
14. git remote add upstream <link of original repository>
15. git push origin <name of the branch in which you want to push>

To pull changes from original repository
16. git pull
